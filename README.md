# Hexafilm.ai Marketing Website

Production static website converted from the Claude Design Component export.

## Files

- `index.html` - production page markup
- `styles.css` - global styles, fonts, animation keyframes, responsive safety rules
- `script.js` - aperture animation, scroll reveal, FAQ accordion, mobile nav, smooth scrolling
- `# Hexafilm.ai Marketing Website/` - original Claude Design export and visual references

## Run Locally

Open `index.html` directly, or serve the folder locally:

```sh
python3 -m http.server 4173
```

Then visit:

`http://127.0.0.1:4173/`

## Deployment

This is a static site and can deploy to GitHub Pages, Netlify, Vercel, or Cloudflare Pages. The `uploads/` folder from the Claude export is ignored and should stay out of GitHub.
